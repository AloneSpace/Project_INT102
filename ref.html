Reference

Property Meaning Sample
@import การนำเข้า CSS แบบ External เมื่อเราเขียน CSS ไปเรื่อยๆไฟล์จะใหญ่แล้วใช้งานยาก เลยต้องเขียนแยกแต่ละตัว @import url('https://fonts.googleapis.com/css2?family=Athiti&display=swap');
transition กำหนดเวลาในการทำงานของ Element transition: ease-out .5s;
transform การปรับตำแหน่งของ Element transform: scale(1.2);
box-shadow การสร้างเงาเพื่อสร้างมิติให้แก่วัตถุ box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.9);ุ
-webkit เป็นเครื่องมือแสดงผลเว็บเบราว์เซอร์ HTML / CSS สำหรับ Safari / Chrome -webkit-transition: background-color 1000ms linear;
grid-template-columns เป็นการแบ่งวัตถุที่คั่นด้วยช่องว่าง แต่ละค่าระบุขนาดของคอลัมน์ที่เกี่ยวข้อง grid-template-columns: 1fr 1fr 1fr;
grid-template-rows เป็นการแบ่งวัตถุที่คั่นด้วยช่องว่าง แต่ละค่าระบุขนาดของแถวที่เกี่ยวข้อง grid-template-rows: 1fr;
grid-template-areas ทำการเพิ่มคุณสมบัติ grid-template-areas ลงในคอนเทนเนอร์กริด โดยคุณสมบัตินี้จะช่วยให้คุณสามารถจัดเรียงเซลล์ในตารางของคุณได้ grid-template-areas: "image" "text" "stats";
grid-area ในเซลล์ตารางและให้แต่ละชื่อแยกกัน ในตัวอย่างจะใช้แท็กชื่อ “header”, “left”, “main”, “right” และ “footer” grid-area: image;
flex-direction ไว้สำหรับบอกว่าเราจะให้ Item มันเรียงยังไง เป็นแถว หรือ เป็นหลัก flex-direction: column;
cursor ช่วยให้เราสามารถใส่รูปแบบให้กับ Cursor ได้หลายแบบ cursor: pointer;
