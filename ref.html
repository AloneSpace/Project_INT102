Reference

Property Meaning
transition กำหนดเวลาในการทำงานของ Element
transform การปรับตำแหน่งของ Element
box-shadow การสร้างเงาเพื่อสร้างมิติให้แก่วัตถุ
-webkit-transition เป็นเครื่องมือแสดงผลเว็บเบราว์เซอร์ HTML / CSS สำหรับ Safari / Chrome
grid-template-columns เป็นการแบ่งวัตถุที่คั่นด้วยช่องว่าง แต่ละค่าระบุขนาดของคอลัมน์ที่เกี่ยวข้อง
grid-template-rows เป็นการแบ่งวัตถุที่คั่นด้วยช่องว่าง แต่ละค่าระบุขนาดของคอลัมน์ที่เกี่ยวข้อง
flex-direction ไว้สำหรับบอกว่าเราจะให้ Item มันเรียงยังไง เป็นแถว หรือ เป็นหลัก
